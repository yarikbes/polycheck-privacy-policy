Privacy Policy for PolyCheck Assistant

Effective date: 2026-02-27

PolyCheck Assistant is a browser extension for SEO and text quality checks.

Summary

  • We do not sell personal data.
  • We do not transfer personal data to third parties for advertising.
  • We do not use personal data for creditworthiness decisions.

What the extension does

PolyCheck Assistant analyzes text selected by the user to provide language and formatting checks, including regional variants, punctuation, number/currency formatting, headings, and readability hints.

Data handling

  • The extension is designed to process text for analysis and reporting within the extension workflow.
  • We do not require user account registration.
  • We do not provide advertising or tracking SDKs in the extension.

Permissions and why they are needed

  • activeTab, scripting: run analysis on the active tab after user action.
  • Host permissions (http://*/*, https://*/*): analyze selected text on pages opened by the user.
  • clipboardRead: user-triggered clipboard fallback when direct selection is unavailable.
  • clipboardWrite: user-triggered copy of report snippets/context.
  • offscreen: supports local processing required by the extension architecture.

Remote code

PolyCheck Assistant does not execute remotely hosted code as part of extension functionality.

Changes to this policy

We may update this Privacy Policy when product functionality changes. The latest version is always published at this URL.
